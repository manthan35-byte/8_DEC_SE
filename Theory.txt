** Client & Server **

[1] Client Program 
    -> Running on end Host
    -> Request Server 
        e.g. Web browser

[2] Server Program
    -> Running on end host
    -> Provides Services
     e.g. Webserver

 // Client - Server Communication //

    [1] Client  - Sometimes on

    -> Intilates a Request to the server when intersted.
      e.g Web Browser on your laptop & cell phone
    -> Dosen't Communication directly with other clients.
    -> needs to know the servers address.

    [2] Server - Always on 

    -> Service requests from many host
    ->  e.g. web server for the WWW.demo.com web site.
    -> doesn't initiate contract with the clients.
    -> needs a fixeed well know address.

** HTTP :- Hyper Text transfer Protocol.
        -> Which is mainly work as a waiter or delivery boy.
        -> Which take request from client and send request to server.
        -> reciver response from server and return back to client.

   HTTPS :- Hyper Text transfer Protocol Secure.
        -> Protocol :- in simple language we can say rule.

VERSION CONTROL SYSTEM : (VCS)

    version CONTROL system mainly track user's record 
    which can track all changes and we can go back to previous changes using of vcs 

here we have one version control tool : git 

git : open source version control software or tool 

github : github is a version control hosting platform using of git we can push / pull 
        data or records on github platform. 

-----------------------------------------------
steps to configure cli git - github 
----------------------------------------------------
(Apply below steps only 1st time at installation)
    step 1 : install git 

    step 2 : open git bash  (windows only)

    step 3 : to check git's version 
        #git --version 

    step 4 : conifgure your github account with git 

            in git bash perform below commands 

            #git config --global user.name "username"
            #git config --global user.email "emailaddress" 

    step 5 : to check your configuration is done or not 

            #git config --list   (you can check your name and email in that list )
