File handling concept in c++

file handling : to accept data from user and store in file 
or we can say that to perform operation on  file 

file operations : 
->create 
-> read 
->write 
->delete 
->update 

file handling class  :

1)fstream 
2)ifstream 
3)ofstream

1)fstream : file stream  -  it is used for both read and write operations

2)ifstream : input file stream  -  it is used for read operations

3)ofstream : output file stream -  it is used for write operations 


file handling syntax : 

#include<fstream>

fstream <filename> <mode>;

file handling modes :

1)ios::in - input mode 
2)ios::out - output mode 
3)ios::app - append mode
4)ios::trunc - truncate mode
5)ios::binary - binary mode
6)ios::ate - at end mode
